## Cloud Top Height by Sentinel-5P TROPOMI (Montly)
Cloud top height (CTH) is a fundamental cloud property describing the altitude of the visible portion of a cloud. 
Accurate knowledge of cloud vertical structure is essential for understanding the Earth's radiation budget, atmospheric dynamics, and the water cycle, as clouds at different altitudes have very different radiative effects. 

#### About the dataset
- The CTH product is measured by the [TROPOMI instrument](https://www.tropomi.eu/) aboard ESA’s [Sentinel-5P Precursor](https://www.esa.int/Applications/Observing_the_Earth/Copernicus/Sentinel-5P) satellite.
- The cloud top height (CTH) product is a montly composite dataset retrieved using the ROCINN (Retrieval Of Cloud Information using Neural Networks) algorithm, which models clouds either as reflecting boundaries (CRB) or as layers with finite optical thickness (CAL).
- More information on the TROPOMI cloud product measurements and quality assessment can be found in the [Product Readme file](https://sentiwiki.copernicus.eu/__attachments/1673595/S5P-MPC-DLR-PRF-CLOUD%20-%20Sentinel-5P%20Cloud%20Level%202%20Product%20Readme%20File%202025%20-%202.10.pdf).

#### Use cases
- Studying the vertical structure of cloud systems and their seasonal variability.
- Monitoring deep convective clouds and their role in atmospheric dynamics.
- Extending the long-term satellite record of cloud properties for climate studies.
